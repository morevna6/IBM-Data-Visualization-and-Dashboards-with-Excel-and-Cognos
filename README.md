# IBM Data Visualization and Dashboards (Excel & Cognos)

## Overview
This repository contains my work from the IBM Data Analyst Professional Certificate module: Data Visualization and Dashboards with Excel and Cognos.

The project focuses on building interactive dashboards to analyze sales performance, customer sentiment, and product-related issues.

## Tools & Technologies
- Excel (Pivot Tables, KPI dashboards)
- IBM Cognos Analytics
- Data Visualization & Dashboard Design

## Key Dashboards

### 1. Sales Overview Dashboard
- KPI tracking: Profit ($78.4M), Quantity Sold (58K+)
- Profit analysis by Dealer ID
- High-level performance monitoring for business decisions

### 2. Service & Customer Insights Dashboard
- Recalls by car model
- Customer sentiment analysis (Positive, Neutral, Negative)
- Monthly quantity sold vs profit (trend analysis)
- Heatmap of recalls by affected system (engineering insights)

## Key Insights
- Certain dealers consistently outperform others in profit generation
- Sales peak mid-year, followed by noticeable drops (seasonality effect)
- Positive sentiment dominates, but negative feedback is still significant
- Recalls are concentrated in specific systems, indicating potential quality issues
